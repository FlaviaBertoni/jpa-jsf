Início sobre o sistema.

Troca Figurinha 2.0 é um sistema desenvolvido como estudo de caso da disciplina de Desenvolvimento de Sistemas Web (DSW) no IFSP - Campus Bragança Paulista com as tecnologias JSF e JPA, sendo utilizado também os componentes do primefaces e ajax.

O sistema é dividido em duas partes:

  • Administrador: possui as funções de cadastro, alteração, busca e exclusão de figurinhas e álbuns.
    •
  • Usuário: possui as opções de criar uma conta, alterar e excluir sua conta, publicar um anúncio de troca de figurinha, excluir seus anúncios e buscar por anúncios de outros usuários.
    •

Esta é a sessão de usuário, é possível acessar todas as funções pelo menu ao lado →.
